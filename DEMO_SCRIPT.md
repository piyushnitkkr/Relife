## Tips for Recording

1. **Practice twice** before recording
2. **Keep mouse movements smooth** — don't rush clicks
3. **Pause 1 second** after each result loads so judges can see it
4. **Speak clearly** — slightly slower than normal conversation
5. **Don't show errors** — if something fails, re-record that segment
6. **End on the dashboard** — strong visual impact for the closing shot
7. **Total should be 3:30-4:00** — judges appreciate conciseness
